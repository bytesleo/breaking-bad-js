## Directory Layout

```bash
.
├── /css/                   # folder styles
│   └── /main.css           # main style
├── /js/                    # folder scripts
│   └── /main.js            # main script
└── index.html              # main html
```

## Installation

```bash
git clone https://github.com/kevoj/breaking-bad-js
cd breaking-bad-js
npm i
```

## Start

`npm start`
